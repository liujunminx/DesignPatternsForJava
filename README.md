## 适配器模式
 *目标(Target):目标是一个接口，该接口是客户想使用的接口
 **被适配者(Adaptee):被适配者是一个已经存在的接口或抽象类，这个接口或抽象类需要适配。
 **适配器(Adapter):适配器是一个类，该类实现了目标接口并包含有被适配者的引用，即适配器的职责是
 对被适配者接口(抽象类)与目标接口进行适配
